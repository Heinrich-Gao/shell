# 🖥️ 批量创建 AWS EC2 实例脚本

## 📌 简介
该脚本基于 **Bash + AWS CLI**，用于 **批量创建 EC2 实例**。  
它会从文件中逐行读取实例名称，检查是否已有同名实例：  
- 如果存在同名实例，则跳过。  
- 如果不存在，则自动创建，并等待实例进入 `running` 状态。  

适合 DevOps 工程师或学习 AWS 的同学，快速批量部署测试环境。

---

## 🔧 功能特点
- 逐行读取输入文件中的实例名称。  
- 检查是否存在同名实例，避免重复创建。  
- 自动为实例打上 `Name` 标签。  
- 自动等待实例进入 `running` 状态。  
- 脚本简洁易懂，便于修改和扩展。
